## Changes

### Added

- Added a Git config editor in Collab for repo and global Git config entries.
- Collab now lets branch and tag selections jump to matching history entries.
- Chat change files now support a tree view and remember the selected list/tree mode.
- Codex model lists now refresh from the account's available models and show supported reasoning efforts.

### Fixed

- Chat now preserves the viewport while tool call groups collapse or expand.
- Improved continuous and nested tool call rendering to reduce duplicate or misplaced tool call records.
- App update checks now read the JSON update manifest, keeping release notes aligned with the in-app update view.
- Improved the initial app loading surface.
- Updated Chinese and English copy for Git config and model selection.
- Reduced folder icon accent colors in Asset views.
