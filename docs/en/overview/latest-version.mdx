## Changes

### Added

- Added an embedded Locus window for Unity Editor under `Window/Locus`, with desktop app detection and launch, window state sync, input focus handling, asset selection sync, and Inspector opening.
- Added the experimental `unity_run_states` tool for running frame-driven C# state machines in Unity to test input flows and capture runtime state.
- Unity asset and scene object references in Markdown now render as interactive references. Click jumps to Unity, and Ctrl+click opens a separate Inspector window.
- `unity_execute` now supports async execution, cross-frame waits, cancellation checks, and progress reporting. Long-running task progress appears in the tool call panel.
- Improved console browsing with frontend/backend log source filters, search, auto-scroll, refresh, and clear actions.

### Fixed

- Fixed NSIS installer upgrade compatibility so updates reuse the previous install location.
- Responses API requests now retry on server errors to reduce transient failures.
- Hardened the asset reference graph watcher to reduce instability during asset database watching and rescans.
- Fixed Unity embedded window focus, mouse activation, window size sync, and Windows frame handling.
- Unity plugin installation now handles DLL conflicts in Unity 2021 projects to reduce compile conflicts after install.
- Fixed missing Print output while `unity_run_states` is running.
- Improved Chat responsive layout, compact input layout, and tool call handoff rendering.
- Improved Diff viewer controls and added fileID disambiguation to Prefab panel titles.
- Fixed Knowledge state reset after workspace switches, search result refresh, and empty folder row alignment.
- Updated Chinese and English UI copy.
