## 变更列表

### 新增

- Collab 新增 Git 配置编辑器，可编辑仓库级和全局 Git config。
- Collab 侧栏支持选择分支和标签，并可跳转到对应历史记录。
- Chat 变更文件新增树形视图，并会记住列表/树形模式。
- Codex 模型列表会从账号可用模型中刷新，并显示支持的 reasoning effort。

### 修复

- 工具调用折叠或展开时，Chat 视口会保持在原位置。
- 改进连续和嵌套工具调用展示，减少重复或错位的工具调用记录。
- 应用更新检查改为读取 JSON 更新清单，发布文档与应用内更新内容保持一致。
- 应用初始加载界面更稳定。
- 更新 Git 配置和模型选择相关中英文文案。
- 降低 Asset 视图中文件夹图标的强调色。
